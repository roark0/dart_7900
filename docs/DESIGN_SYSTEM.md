## 设计系统概述

### 设计理念

该设计系统源自 **L-J 血液分析仪** 的 5 个核心界面截图的视觉分析：

- **Analysis**（分析结果）
- **L-J QC**（质量控制）
- **List Review**（样本列表）
- **Add Diluent**（添加稀释液）
- **Maintenance**（设备维护）

### 核心原则

1. **临床可读性** — 医疗数据优先，清晰的对比度和字体尺寸
2. **工业设计** — 扁平设计，无渐变，高对比度蓝色系
3. **触控友好** — 按钮最小尺寸 44×44px，间距充足
4. **低分辨率适配** — 固定 800×600 屏幕尺寸，字号集中在 11-13px
5. **一致性** — 统一的导航、表格、表单规范

## 字体排版

### 字体系统

**主字体** — 系统无衬线字体（Arial / Segoe UI）  
**等宽字体** — 数据数值显示（Courier New / monospace）  
**回退字体** — Geist, Geist Fallback（Next.js 内置）

### 字体规范表（6 个层级）

| 层级 | 字号 | 字重 | 行高 | 背景色 | 文字色 | 用途 | 示例 |
|------|------|------|------|--------|--------|------|------|
| **T1** — 导航标签 | 13px | 700 Bold | 1.2 | `#1a4f7a` | `#ffffff` | 顶部主导航Tab标签文字 | Analysis, List Review, L-J QC |
| **T2** — 区域标题 | 12px | 600 SemiBold | 1.3 | `#3a8fbf` | `#ffffff` | 表格列标题行、侧边栏菜单标签 | Item, Result, Unit, WBC Flag |
| **T3** — 参数名称 | 12px | 500 Medium | 1.4 | `#ffffff` | `#1a5276` | CBC参数名称（正常状态为深蓝色） | WBC, Lym%, Mid%, Gran% |
| **T3A** — 参数名称（异常） | 12px | 500 Medium | 1.4 | `#ffffff` | `#d4820a` | 超出参考范围的参数名称 | MCV, MCH, MCHC, RDW-CV |
| **T4** — 数据数值 | 11px | 400 Regular | 1.5 | `#ffffff` | `#1e2d3a` | 检测结果数值、单位值 | 5.23, 3.89, 132, 10⁹/L |
| **T5** — 状态栏 | 11px | 400 Regular | 1.2 | `#0d3a5c` | `#d0e8f5` | 底部状态栏操作员、时间戳 | Engineer, 2026-03-18 15:56:10 |
| **T6** — 表单标签 | 11px | 600 SemiBold | 1.4 | `#e8f2f8` | `#1a4f7a` | 表单字段标签、配置项名称 | FileNo., LotNo., Exp.Date |

### 字体排版规则

1. 主字体使用系统无衬线字体（Arial / Segoe UI），不引入额外字体文件，确保医疗设备环境兼容性
2. 数值类数据优先使用等宽字体显示，确保小数点对齐一致
3. 所有文字最小字号为 11px，确保 800×600 低分辨率屏幕可读性
4. 异常参数使用橙色 `#d4820a` 加粗，正常参数使用深蓝 `#1a5276` 区分
5. 白底区域文字使用深色 `#1e2d3a`，蓝底区域文字使用白色 `#ffffff`
6. 表单标签左对齐，数值右对齐，参数名左对齐，保持视觉统一

## 布局规范

### 屏幕布局（800×600 固定）

```
┌─────────────────────────────────────────────┐
│  [导航栏] 深蓝 #1a4f7a ← 固定高度 64px    │
│  Analysis | List Review | L-J QC | ... │
├─────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────┐ │
│ │                                       │ │
│ │      主内容区（内边距 16px）         │ │
│ │     浅蓝背景 #e8f2f8                 │ │
│ │                                       │ │
│ └─────────────────────────────────────────┘ │
├─────────────────────────────────────────────┤
│  [状态栏] 深蓝 #0d3a5c ← 固定高度 24px    │
│  Engineer | 2026-03-18 15:56:10 |  ℹ  │
└─────────────────────────────────────────────┘

总高度: 600px = 导航 (64px) + 内容 (512px) + 状态栏 (24px)
```

### 内容区布局方法

**优先级**：

1. **Flexbox** — 绝大多数布局（导航、表单行、按钮组、表格行）
2. **CSS Grid** — 复杂 2D 布局（参数表格、QC配置表）
3. **避免** — Float、绝对定位（除非必要）

### 左侧竖向菜单（侧边栏）

- **宽度** — 100px 固定
- **按钮** — 白文字，深蓝背景 `#2d6fa0`，高度 40px，间距 2px
- **选中态** — 背景改为 `#3a9bd5`，保留白文字
- **圆角** — 4px

### 主内容区

- **内边距** — 16px（顶左右下）
- **背景色** — 浅蓝 `#e8f2f8`
- **最大宽度** — 700px（中等屏幕响应式）

### 表格网格间距

- **行高** — 28px（包含内边距 8px 顶底）
- **列间距** — 边框 1px，内边距 8px 左右
- **最小列宽** — 60px（参数名）

## 核心界面

### 1. Analysis（分析结果）

**布局结构**：

- 顶部：主导航栏
- 内容区分 4 部分：

```css
@theme inline {
  --color-primary: oklch(0.32 0.1 240);      /* 深海军蓝 */
```
